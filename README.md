# ✕ Tic Tac Toe ○

A modern, interactive Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Demo:** [your-project.vercel.app](https://your-project.vercel.app) <!-- Replace with your Vercel URL after deployment -->

---

## Features

- **Player vs Player** — Two players take turns on the same device
- **Player vs AI** — Challenge a computer opponent
  - **Easy mode** — AI plays random moves (great for beginners)
  - **Hard mode** — AI uses the Minimax algorithm with alpha-beta pruning (unbeatable)
- **Live Scoreboard** — Tracks wins for X, wins for O, and draws across rounds
- **Reset Scores** — Clear the scoreboard without leaving the page
- **Responsive Design** — Works on desktop and mobile browsers
- **Smooth Animations** — Cell pop, score bump, and winning cell pulse effects

## Getting Started

### Run Locally

No build tools or dependencies required. Simply open the file in your browser:

```bash
# Option 1 — open directly
open index.html

# Option 2 — serve with VS Code Live Server or any static server
npx serve .
```

### Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your repository
4. Leave all settings as default (Vercel auto-detects static sites)
5. Click **Deploy**
6. Update the **Live Demo** link at the top of this README with your Vercel URL
